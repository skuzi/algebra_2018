05) Оценка на собственные числа ограничения. Оценка на след.\\
Расписать аккуратно КФ $\lambda_{i+n-m}, \lambda_i$ для $A$, $\mu_i$ для $B$ зажато между ними. След формы - след матрицы в о.н. базисе, не меняется при смене базиса, $q(u_i) = e_i A e_i^{\top}$ = A_{ii}$ ($\tr (AB) = \tr(BA)$). След ограничения не больше следа оригинала.
